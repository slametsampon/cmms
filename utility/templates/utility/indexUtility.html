Utility

Agar system CMMMS ini bisa berjalan dengan baik, diperlukan beberapa konfigurasi meliputi flow chart Work Order serta keterkaitan pada masing-masing stack holder. Termasuk didalamnya perlu dilakukan pengaturan kewenangan masing-masing pengguna (user) dalam kaitanya jalur approval, serta setup lain yang diperlukan. Disamping itu penentuan status Work Order serta tindakan terhadapnya juga perlu didefinisikan. Dalam module ini berfungsi sebagai berikut :

  • Setup organisasi (Departement, Section)
    • Nama
    • Description
    • Initial
    • Role (Originator, Executor)
  • Setup user (flow chart approval)
    • Username
    • Initial
    • Forward/Reverse to
    • Setup organisasi
